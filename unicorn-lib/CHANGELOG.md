# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-15

### Added
- Initial beta release
- **Button component** with variants (primary, secondary, danger) and sizes (sm, md, lg)
- **Card component** with variants (default, elevated, outlined)
- Full TypeScript support with strict mode
- Type definitions (.d.mts) included
- Optimized ESM bundle via VitePlus/tsdown
- Comprehensive documentation and examples
- Vue 3.5.30 as peer dependency
- Development guidelines and component structure

### Features
- Tree-shakeable imports
- Small gzipped bundle size (~1KB)
- Zero external dependencies (Vue is peer dependency)
- Composition API compatible
- Type-safe component props with PropType support

### Setup
- VitePlus configured with tsdown bundler
- TypeScript strict mode enabled
- Vitest ready for component testing
- GitHub Actions ready for CI/CD
